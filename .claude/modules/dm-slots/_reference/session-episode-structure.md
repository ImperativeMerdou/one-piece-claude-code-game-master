# Session Episode Structure — Every Session is an Anime Episode

**The single most-impactful structural rule for feeling like an anime.** Every session has the SHAPE: title card → optional cold open → A-block → eyecatch → B-block → cliffhanger → NEXT TIME ON. Procedural, not optional.

**Division of labor (this module owns the SHAPE):**
- `_reference/anime-craft.md` — Toei craft (theme-song callbacks, sakuga, anime-pacing tricks)
- `_reference/manga-craft.md` — Oda / manga-chapter craft (chapter hooks, callback density, panel rhythm, SBS trivia)

---

## SESSION OPEN

### 1. PREVIOUSLY ON (optional, brief)

If the previous session ended on a cliffhanger, open with 2–4 sentence recap, anime-style, ending on the most intense beat from last session. Skip when the player just played and immediately starts another.

### 2. EPISODE TITLE CARD (mandatory)

Format: **"[hook phrase]! [theme statement]!"**

Canon: "OPENING THE GATE! FOR ROBIN!" · "ZORO'S NEW SWORD! WANO'S GREATEST SWORDSMAN!" · "BAJRANG GUN! THE STRONGEST FIST OF GEAR FIFTH!"

```
╔═══════════════════════════════════════════════════════════╗
║              ⛵ EPISODE 3 — DAY 11                        ║
║       "TWO ON THE DOCK! THE CAPTAIN'S CHOICE!"            ║
╚═══════════════════════════════════════════════════════════╝
```

Appears ONCE at session open, referenced ONCE in NEXT TIME ON.

### 3. COLD OPEN (optional)

~5–10 lines before the title card. Meanwhile-cut to villain · sensory anchor · flashback fragment · NPC moment. Often a hook that pays off later.

---

## A-BLOCK / B-BLOCK / EYECATCH

Anime rhythm: A-block (first half) → eyecatch (commercial break) → B-block → ending theme. In TTRPG: at a natural midpoint, drop an EYECATCH. Optional.

```
═══════════════════════════════════════════════════════════
                      ⚓ ⚔ ⚓
═══════════════════════════════════════════════════════════
```

Or with a one-line character beat:
```
─────── EYECATCH ───────
  Sumi sits on the dock crate, drawing
  Merdo from memory. The drawing has
  too many horns.
─────────────────────────
```

---

## DETECTING SESSION END

Fire CLIFFHANGER + NEXT TIME ON when:
1. **Explicit save:** `/dm save` / `/dm end`
2. **Verbal wrap:** "let's end here" / "wrap" / "I gotta go" / "stopping for the night"
3. **Natural stopping action:** Merdo going to sleep, crew docking for the night, an arc closing
4. **Player-offered:** DM OFFERS — *"This is a strong stopping point — want to end on a cliffhanger here?"*

None triggered → session is ongoing. Never force a wrap mid-action; finish the current beat THEN cliffhanger.

---

## CLIFFHANGER — mandatory at session end

Never "and then they went to sleep." Pick one of five shapes:

1. **Arrival** — silhouette in the doorway. News Coo on the railing. A figure the player can't yet identify.
2. **Declaration** — "Captain. We need to talk." / "I know what you are." / "He's coming."
3. **Mid-action cut** — punch thrown, impact next session. Bell rings. Sword comes up. CUT.
4. **Revelation** — intel surfaces in the last sentence. Wax seal. Name on the file. Face from age fifteen.
5. **Question** — scene ends on a question with no answer.

Example:
```
"Captain. There's a name been added to your bracket
since this morning."

She turned the ledger. Merdo's eyes found the entry.

The name was his own father's.
```

---

## NEXT TIME ON — mandatory

4–7 lines. 2–3 dramatic teaser lines + one closing hype line. Teases 2–3 specific beats, names ONE character or object, ends with the next episode title.

```
═══════════════════════════════════════════════════════════
              ⚓ NEXT TIME ON THE MERDO PIRATES ⚓
─────────────────────────────────────────────────────────────

  The first bell rings. The crowd inhales.
  Ezrad raises the bottle. Merdo plants his foot.

  And in a north VIP box, a man with a gold ring
  whispers something the steward of the Tessara Cup
  has never been allowed to hear.

  EPISODE 4: "FIRST BLOOD! THE LAST BOTTLE TOAST!"

═══════════════════════════════════════════════════════════
```

DM doesn't know next session yet → focus on what the player ALREADY KNOWS is coming. Evocative-but-vague.

---

## END-CARD STINGER (rare, 1 per 3–5 sessions)

After NEXT TIME ON, occasional post-credits stinger. ~2–4 lines. Hints at a FUTURE beat, not the next session. Rarity is what makes it land.

---

## Typical session shape

```
[PREVIOUSLY ON] (if continuing from cliffhanger)
[COLD OPEN] (optional)
[EPISODE TITLE CARD]
[HUD: scene state, location, time, bounty, Haki]
[A-BLOCK]
[EYECATCH] (optional)
[B-BLOCK]
[CLIFFHANGER]
[NEXT TIME ON]
[END CARD STINGER] (rare)
```

The shape itself does the work. When every session has it, the user isn't playing a TTRPG — they're watching an episode.
