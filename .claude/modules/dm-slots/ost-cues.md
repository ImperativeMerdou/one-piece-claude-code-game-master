# OST Cues — The Soundtrack-Feel in Prose

**The One Piece anime soundtrack is iconic.** Specific tracks score specific scene types — "Overtaken" for serious peaks, "Difficulty" for emotional struggle, "We Are!" for triumphant moments, "Bink's Sake" for melancholy departures, "Mother Sea" for tender arrival, "Soundless Voice" for grief, "The World" for dramatic arrival. When a fan hears the opening notes of any of these, the emotional context is INSTANT.

Prose can't play music. But it can REFERENCE the soundtrack feel via brief italic scene-direction notes that EVOKE the same moment. This module gives the DM a small toolkit for doing it.

---

## How to deploy — italic scene direction

A track-cue is a single italic line, usually at the START of a peak moment, that reads like a director's note. NOT exposition. NOT narration. A scene-direction.

```
*[the OST starts the way it always starts when this matters]*

Merdo planted his foot.
```

Or with a track-name reference (sparingly, when a fan would catch it):

```
*[Overtaken, slow and rising]*

The bell tolled the second time.
```

The track-cue is OPTIONAL flourish. Use 0-2 times per session, only at peak moments. Overuse kills it.

## Track-feel taxonomy (what to invoke when)

These are the broad EMOTIONAL CUES, mapped to prose moments. Use the FEEL — don't lecture about which track plays.

### THE TRIUMPHANT BUILD ("We Are!" / "Hikari E" / "Wir Sind!")
Use for: arc-end victories, crew unity moments, declarations of will, public defiance.

```
*[the theme returns — that one — the one you know]*

Merdo lifted Danzai. The crowd lifted with him.
```

### THE SERIOUS PEAK ("Overtaken" / "Difficulty")
Use for: power-ups, finishers, ideology fights, moments of will-against-impossibility.

```
*[Overtaken, that slow rising hum]*

Hudore's leash dropped.
```

### THE GRIEF MOMENT ("Soundless Voice" / "Memories" / "Going Merry")
Use for: deaths, departures, character backstory peaks, quiet aftermath of loss.

```
*[the slow piano line — the one that always plays when it ends]*

Brenne closed the Ship Log on the page that mentioned Rook.
```

### THE TENDER ARRIVAL ("Mother Sea" / "Bink's Sake")
Use for: returning home, finding lost family, quiet triumph after a long arc.

```
*[Bink's Sake, sung slow — the way the old sailors sing it]*

The Goroa harbor was visible by the third hour. Mira Odinu
was on the pier.
```

### THE DRAMATIC ARRIVAL ("The World" / "Wave of Rebellion")
Use for: a major character entering a scene, a Yonko moving, a Warlord arriving.

```
*[the brass lifts]*

Garp stood up in the back of the colosseum.
```

### THE COMEDY THEME ("Joke Soundtrack" / Buggy's Theme)
Use for: pure comedy beats, slapstick, the Boot War.

```
*[the silly horn — that one]*

Volo demanded witness.
```

### THE TENSION BUILD ("Wano-Country" / "Ominous")
Use for: stake-rising moments, a villain entering range, the wait before action.

```
*[the low rumble]*

The black ship raised a single signal lantern.
```

---

## Format conventions

- Italic, brief, scene-direction style
- Optional name-reference (when the user would recognize it — fans love this)
- ONE per peak moment maximum
- Goes at the START of the moment, not in the middle
- Uses brackets: `*[...]*`
- Skip entirely on routine scenes — these are reserved for emotional weight

---

## Examples in context

### A finisher
```
The crowd inhaled.

*[Overtaken — the fast version]*

Merdo planted his foot.

**"DANZAI: TOMB-MARKER!!"**

The pit cracked open.

*[the music cuts]*

Silence.
```

### A grief moment
```
The Twenty stood at the rail. Brenne held the Ship Log.

*[the piano starts — the way it always starts when it ends]*

Rook walked down the gangplank with one case and an empty
tea cup. He did not look back.

The crew did not call him.

*[the music carries]*
```

### A comedy beat
```
*[the silly horn — that one]*

Sumi pointed at Merdo. "I'M JOINING YOUR CREW!"

*[the silly horn loops, then cuts]*

Mako put his face in his hands.
```

---

## When NOT to use

- Routine dialogue / exploration scenes — no need
- Already-loud moments where the prose carries the energy — don't pile on
- Comedy that doesn't need the meta-flourish (most comedy)
- More than 2x per session — saturation kills the device

---

## Why this works

Real anime episodes have specific tracks SCORING specific moments. Fans associate the music with the scene. When the track starts, the audience KNOWS what kind of beat is coming.

Prose can't trigger the music. But the user IS A FAN. When the DM writes *[Overtaken — the fast version]* — the user HEARS it in their head. That's the device working. The reader supplies the audio.

This is a pleasure-flourish. It rewards the user for being a One Piece fan. It's optional but iconic.

## Track-name vocabulary (memorize these — fans recognize them)

Major:
- "We Are!" — opening theme, triumph
- "Bink's Sake" — pirate folk song, melancholy / departure
- "Overtaken" — serious peak, finisher buildup
- "Difficulty" — emotional struggle
- "The World" / "The World ~ Episode of Luffy ~" — dramatic arrival
- "Mother Sea" — tender / nostalgic
- "Soundless Voice" — grief / parting
- "Memories" — flashback / soft sad
- "Wave of Rebellion" — uprising / war
- "One Day" — quiet courage
- "Sailing" — voyage / movement
- "Wano-Country" — exotic / tension
- "Difficulty" — character struggle
- "Adventure World" — childlike wonder, new arrival

Refer to tracks by NAME occasionally. Refer by FEEL more often. Either works.

---

## What this buys the campaign

A pleasure-flourish that makes the user feel like he's watching the show. Optional, sparing, never required — but when it lands, it lands hard. The right cue at the right beat is anime alchemy.
