# Manga Panel Grammar — Write Scenes as Panels, Not Paragraphs

**Manga is read panel by panel.** Each panel has ONE focused content. Multiple panels stack to make a page. The reader's eye moves: panel → panel → panel → BIG IMPACT PANEL → page-turn-reveal.

When prose is written like a novel — flowing paragraphs of inter-related sensory detail — it loses the manga rhythm. The reader doesn't get the panel-by-panel beats. The peaks become indistinguishable from the troughs.

This module teaches the DM to write One Piece scenes as PANELS.

---

## THE PANEL TYPES (Scott McCloud + Oda variants)

### Type 1: WIDE SHOT
Establishes location, scale, scope. One sentence describing what the camera sees from far back.

```
The colosseum was eight thousand voices climbing one wall of sound.
```

### Type 2: MID SHOT
Multiple characters in frame, action / dialogue happening. 1-3 sentences.

```
Partho the bookie waved a chalk hand at Merdo across the floor.
The crowd parted around him without thinking.
```

### Type 3: CLOSE-UP
One face, one detail, one body part doing one thing. 1 sentence usually.

```
Sumi's mouth was open. She had not closed it since the bell.
```

### Type 4: REACTION SHOT
A character's face responding to what just happened. NPC or witness, named or unnamed. 1 sentence.

```
A merchant near the railing dropped his roasted skewer.
```

### Type 5: SOUND-EFFECT PANEL
A panel that's mostly the SFX. The image is implied through the sound. Often single word, ALL CAPS, on its own line.

```
**DON.**
```

### Type 6: SILENCE PANEL
A beat of nothing happening. Manga uses **shiin (シーン)** — the symbol for silence. In prose, a single brief sentence or single italic word.

```
Silence.
```

Or: a description of stillness with no action verb.
```
The pit was empty of sound.
```

### Type 7: SPLASH PANEL
The big-impact panel that takes up a full page (or in prose: a paragraph break before AND after, plus a single weighted sentence).

```
He lifted Danzai over his head.

The colosseum stopped.

**"DANZAI: TOMB-MARKER!!"**

The pit floor cracked from one wall to the other.
```

The splash panel is a SCREAM moment, a finisher, a reveal. Use sparingly — the rarity makes it matter.

## PANEL TRANSITIONS (Scott McCloud's six)

How does the reader's eye move from one panel to the next? Each transition does different work.

### 1. Moment-to-moment
Two panels showing tiny progression. Use for slowing time at peak moments.
```
The bell tilted.

The bell rang.
```

### 2. Action-to-action
The standard progression. Subject does one thing, then the next thing.
```
Merdo planted his foot. The gauntlet came up.
```

### 3. Subject-to-subject
Camera moves to a different subject in the same scene. Use for crowd reactions, parallel action.
```
Merdo planted his foot.

In the north VIP box, Yann set down his teacup.
```

### 4. Scene-to-scene
Cut to different location/time. The biggest jump. Use for meanwhile cuts.
```
The bell rang.

═══ MEANWHILE — DRESSROSA, THE COMMAND ROOM ═══
```

### 5. Aspect-to-aspect
Camera moves around the SAME moment, capturing different details. Slows time. Beautiful for setting peak moments.
```
Sumi's mouth open.

The dust settling on Ezrad's bottle.

A cat on Ashren's boot, eyes half-open.
```

### 6. Non-sequitur
Two panels with no logical connection. Almost never use — only for explicit absurdism / dream beats.

## WRITING IN PANELS — practical rules

### Rule 1: One panel = one focus
Don't pack three things into one paragraph. If you have three things, that's three panels. Three short paragraphs.

WRONG (novel-style, all in one):
> The crowd surged forward as Merdo planted his foot, and Oleg spat over the rail without looking up while Sumi screamed encouragement from the railing and Ezrad raised his bottle in a salute that might have been a toast or might have been a goodbye.

RIGHT (manga-style, panel by panel):
> Merdo planted his foot.
>
> The crowd surged forward.
>
> Oleg spat over the rail. Didn't look up.
>
> Sumi screamed something from the railing.
>
> Ezrad raised the bottle. It might have been a toast. It might have been a goodbye.

### Rule 2: Vary panel types per beat
Don't write five wide shots in a row. Stack: wide → mid → close → reaction → impact. The variation IS the rhythm.

### Rule 3: Use SHORT panels for fast beats, LONG panels for slow beats
- Combat exchange: short panels (1 sentence each, action-to-action)
- Quiet contemplation: longer panels (sensory-rich, aspect-to-aspect)
- Crisis: alternating (short → silence → short)

### Rule 4: Page-turn-reveal moments
The most powerful manga device: the reveal happens AFTER the page turn. In prose, this is a single sentence on its own line, after a break, that recontextualizes everything.

```
Merdo's eyes found the entry on the bracket sheet.

The name written in was his own father's.
```

The reveal sentence is short. The break before it is the page turn.

### Rule 5: NO dialogue tags during panel-grammar action
"He said" / "she said" interrupts the panel rhythm. Let the dialogue stand alone.

```
"DANZAI: TOMB-MARKER!!"

The kanabo came down.
```

NOT:
> Merdo said, "DANZAI: TOMB-MARKER!" as he brought the kanabo down.

### Rule 6: Match paragraph break to panel break
If two beats are in two panels, they're in two paragraphs. Don't merge.

## CANON ODA EXAMPLES TO REFERENCE

### Robin's "I want to live!" (Enies Lobby chapters 397-398)
The page is mostly silence panels, then the BIG splash of Robin screaming. The contrast between the quiet panels and the splash is the entire emotional payoff.

### Luffy vs Lucci finisher (Enies Lobby chapter 427)
The "Jet Gatling" page-turn: small panels of build-up, then page-turn → SPLASH → silence. Then aftermath.

### Zoro vs Mr. 1 "Ittoryu Iai: Shi Shi Sonson" (Alabasta chapter 195)
A single splash panel of the cut. Then silence panels of the aftermath. Then Mr. 1 falling.

### Wano "Bajrang Gun" (chapter 1049)
The build is many small panels. The fist itself is multiple full-page splashes. The aftermath is silent civilians celebrating.

The PATTERN: build with small panels → splash for impact → silence for aftermath.

## WHEN TO USE PANEL GRAMMAR

- **Action sequences** — always
- **Big reveals** — always
- **Emotional peaks** — always
- **Comedy beats with multiple reactions** — always
- **Quiet contemplative moments** — sometimes (use long panels — wide / aspect-to-aspect)
- **Routine narration / setup** — no, just write naturally

The point is NOT to write every scene as panels. The point is to KNOW when you're in panel-mode and execute it.

---

## WHEN NOT TO USE PANEL GRAMMAR

- Routine dialogue exchanges between two characters (just dialogue, with action beats)
- Player-driven exploration ("I look at the basket") — respond conversationally
- Inner monologue / quiet contemplation that's better as flowing sentences
- When the player is asking an OOC question — answer plainly

Panel grammar is a TOOL for specific moments. Not a default for everything. The DEFAULT is conversational prose with sensory specificity. Panel grammar fires when the SCENE'S WEIGHT calls for it.

## PUTTING IT ALL TOGETHER

A peak fight beat in panel grammar:

```
The bell rang.

Ezrad smiled.

Merdo did not.

Two cleavers moved.

"Hah-hah-HAH —"

**DON.**

The first cleaver caught the rail an inch from Merdo's hip.

The second was already coming for his throat.

Merdo planted his foot.

═══════════════════════

He stopped looking at Ezrad and looked at the bottle.

The bottle was full.

═══════════════════════

**"BJORN."**

The Bear Stance dropped.
```

Each line is a panel. The breaks are page-turns. The named-attack invocation is the splash. The looking-at-the-bottle aspect-shot is the slow beat that lets the reader breathe before the next exchange.

THIS is reading manga.
